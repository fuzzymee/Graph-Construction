With the MainMST.java, Kruskal.java, UpTree.java, Heap.java, AdjacencyList.java, Vertex.java,
EdgeRecord.java and input files in the same directory run the program with the following command line:

compile: eos$ javac MainMST.java Kruskal.java UpTree.java Heap.java AdjacencyList.java Vertex.java EdgeRecord.java
run: eos$ java MainMST < [input file] > [output file]
